
## 语义理解/口语理解
>>> 采用了深度学习等技术

主要功能：
* 中文分词
* 词性标注
* 命名实体识别
* 领域分类
* 槽填充
* 意图识别

自然语言处理使用的是作者开源的Jiagu项目：https://github.com/ownthink/Jiagu

spoken举例：
```
上海的天气
厦门明天会不会下雨
姚明真正的身高是多少
```
